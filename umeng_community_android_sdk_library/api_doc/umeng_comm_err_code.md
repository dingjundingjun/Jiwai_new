# Android SDK内部错误码


|  错误码  |       含义     |
|---------|---------------|
|    -1   | 未知错误,通常为执行网络请求失败 |
|     0   | 微社区的请求无错误  |
|    200  | 微社区集成的其他SDK的网络请求成功 |
|    -101  | 连接错误，通常是无效url引发的网络请求失败 |
|    -102  | 网络请求超时 |
|    -103  | 没有网络连接 |
